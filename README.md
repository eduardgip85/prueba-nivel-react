# Mi App

Prueba de nivel react


## Objetivos con esta prueba
Objetivo: Evaluar la capacidad del desarrollador para construir una aplicación React funcional con TypeScript, gestionando una colección de datos local y navegación entre componentes, utilizando gestión de estado moderno (Hooks) y siguiendo las mejores prácticas de arquitectura (Custom Hooks, Componentes y React Router).


## Descripcion del proyecto
Lista de productos donde se pueden ver al detalle con una pantalla de detalle de cada producto



## Requisitos previos
- Node.js (versión 16 o superior)
- npm (viene incluido con Node.js)


## Instalación
- npm install
- npm run dev

Para hacer los test
- npm run test / npx vitest run

## Tecnologias usadas
- React 19+
- Typescript
- React router
- Vite
- Vitest
- Tailwind
- Testing library
